# Pokédex Reimagined

A responsive React-based web application that allows users to explore and view detailed Pokémon data through an interactive interface. The application fetches real-time data from the PokeAPI and presents it using dynamic UI rendering and client-side routing.

**Live Website:**  
https://03ec0551.pokedex-reimagined.pages.dev/pokemon/charmeleon  

## Features
- All availible pokemon are dispalyed
- Click on their Icons or Search for a pokemon with their name or Dex entry number to get their details
- Turn on Shiny mode to see all shiny forms of the pokemon
- Search for a pokemon you know to find their Details

## Dependencies
- React Router DOM
- Tailwind CSS
